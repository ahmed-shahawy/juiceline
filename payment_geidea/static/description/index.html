Geidea Payment Provider for Odoo 18 This module provides integration with Geidea payment gateway for Odoo 18. Features: - Credit card payments through Geidea - Support for multiple Middle Eastern currencies - Compatible with Odoo 18 payment provider framework - Secure tokenization support Installation: 1. Place this module in your Odoo addons directory 2. Update the module list 3. Install the "Geidea Payment Provider" module 4. Configure your Geidea credentials in the payment provider settings
